# 数据分类与处理基线

| 等级 | 示例 | 存储与访问 | 日志/导出/处置 |
|---|---|---|---|
| C1 公开 | 产品公开说明、非敏感字典 | 普通完整性保护 | 可记录，按版本保留 |
| C2 内部 | 项目一般信息、任务、配置元数据 | 工作空间和角色授权，传输加密 | 日志仅稳定标识；导出需授权 |
| C3 机密 | 合同引用、审批意见、交付件、客户信息、成本预留字段 | 对象/字段权限、存储加密、下载复核 | 禁止正文日志；导出审批、短时地址和水印策略 |
| C4 严格机密 | 凭据、密钥、签名材料、设备凭据、迁移原始数据、生产导出 | 外部密钥引用、最小权限、独立账号与隔离存储 | 永不进入仓库和应用日志；到期安全处置并保留审计 |

数据对象必须记录分类、责任工作空间、保留策略和法律保留状态。分类继承可被更严格规则覆盖，不得降级绕过字段、附件、导出和迁移控制。测试数据必须脱敏或合成。
